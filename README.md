# Signal Lab: Predicción de Riesgo de Diabetes Tipo 2

Este repositorio contiene el desarrollo de un modelo de Inteligencia Artificial para el tamizaje preventivo de diabetes, basado en la **ENSANUT Continua 2024**.

## Ver Proyecto (Recomendado)
Para una experiencia visual de los resultados y la metodología, visita nuestro sitio oficial:
**[https://byf1nd.github.io/deteccion-diabetes-ia/](https://byf1nd.github.io/deteccion-diabetes-ia/)**

## Contenido del Repositorio
- `notebooks/`: Código fuente en Python (Google Colab).
- `data/`: Datasets procesados y base de datos final.
- `Reporte_Final.pdf`: Documentación técnica completa.
